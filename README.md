# ARCANIS V0

ARCANIS est un simulateur de prise de decision professionnelle pour consultants QSE.

## Lancer en local

```bash
pnpm install
pnpm dev
```

## Verifier le projet

```bash
pnpm typecheck
pnpm lint
pnpm build
```

## Deploiement

Le projet est pret pour Vercel avec Next.js App Router, TypeScript et Tailwind CSS.
